# statistik_och_dataanalys_kunskapskontroll_2

This repository contains assignments from a course in Statistics and Data Analysis.

The project focuses on analyzing raw bank data, starting with data cleaning and preprocessing (see `data_cleaning`). A machine learning workflow was then applied using clustering to segment customers into active and passive groups based on key behavioral features.

The final clustered dataset was exported to Excel and used to build visualizations and insights in Power BI.
